# 🛒 Amazon Clone

A responsive Amazon Clone built using only HTML and CSS.

This project is a frontend UI clone of Amazon's homepage created to practice real-world website layouts, and modern CSS techniques.

## 🚀 Features

✅ Amazon-inspired Homepage Design  
✅ Navigation Bar  
✅ Search Bar UI  
✅ Product Sections  
✅ Hero Banner  
✅ Footer Design    
✅ Clean and Structured Code

## 🎯 What I Learned

Through this project, I improved my understanding of:

- HTML Structure
- CSS Flexbox
- CSS Grid
- Website Layout Creation
- UI Cloning Techniques
